本研究针对我国康复医疗服务资源分布不均、专业人才短缺及慢病高发等问题， 提出并构建了一种基于人工智能驱动的健康评估与干预一体化平台。平台集成多 模态人体姿态捕捉，通过大规模预训练模型与知识图谱技术，将专家经验结构化为 可迁移的运动处方模型，实现了“AI 持续进化—医师精准干预”的闭环体系。具体 方法：1.数据采集：利用Kinect体感设备，实时采集患者运动信息；2.知识图谱构 建：融合专家临床方案、文献与实践案例，形成多维度康复医学图谱；3.模型训练 与迭代：基于小样本学习与动态置信度评估机制，对复杂并发症进行高达 82%的 识别准确率，实现智能审核屏障；4.个性化运动处方生成：根据患者术后状态及体 质特征，推荐针对性运动处方，并通过强化学习不断优化训练方案。
